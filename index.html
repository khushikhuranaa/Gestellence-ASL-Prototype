Gestellence

"AI-Based Hand Gesture and Voice to Visual Communication System: Bridging the communication gap between hearing-impaired and hearing individuals through intelligent, real-time translation of sign language and speech."

Problem Statement

Communication between hearing-impaired and hearing individuals remains constrained due to the lack of mutual understanding of sign language and the absence of an intelligent, assistive tool capable of recognizing continuous sign-language gestures and multilingual speech and converting them into text or animated sign output.

Literature Review / Market Research

Analyzed approaches including CNN-based sign language recognition, MediaPipe for hand tracking, transformer-based translation frameworks, and 3D avatar systems for Indian Sign Language. Existing systems focus on isolated word detection, limiting real-time continuous communication efficacy.Most existing systems achieve high accuracy in controlled environments but struggle with real-world variability such as lighting conditions, background noise, and continuous signing.

Research Gap / Innovation

Our system integrates real-time gesture recognition, multilingual speech-to-text, and 3D avatar-based text-to-sign translation into a single unified web platform. Unlike existing systems that operate independently, our approach enables seamless continuous communication in both directions.

Real-time webcam video for hand gesture detection using OpenCV. MediaPipe extracts 21 hand landmarks per frame at 30 FPS. Audio input captured for multilingual speech recognition.

                    import cv2
import mediapipe as mp

mp_hands = mp.solutions.hands
mp_draw = mp.solutions.drawing_utils

cap = cv2.VideoCapture(0)

def detect_gesture(landmarks):

    thumb_tip = landmarks[4]
    index_tip = landmarks[8]
    middle_tip = landmarks[12]
    ring_tip = landmarks[16]
    pinky_tip = landmarks[20]

    index_base = landmarks[6]
    middle_base = landmarks[10]
    ring_base = landmarks[14]
    pinky_base = landmarks[18]

    #  STOP (Open Palm)
    if (index_tip.y < index_base.y and
        middle_tip.y < middle_base.y and
        ring_tip.y < ring_base.y and
        pinky_tip.y < pinky_base.y):
        return "STOP"

    #  YES (Closed Fist)
    if (index_tip.y > index_base.y and
        middle_tip.y > middle_base.y and
        ring_tip.y > ring_base.y and
        pinky_tip.y > pinky_base.y):
        return "YES"

    #  NO (Two Fingers Up)
    if (index_tip.y < index_base.y and
        middle_tip.y < middle_base.y and
        ring_tip.y > ring_base.y and
        pinky_tip.y > pinky_base.y):
        return "NO"

    #  CALL ME (Thumb + Pinky Up)
    if (thumb_tip.y < index_base.y and
        pinky_tip.y < pinky_base.y and
        index_tip.y > index_base.y and
        middle_tip.y > middle_base.y and
        ring_tip.y > ring_base.y):
        return "CALL ME"


    #  OK SIGN
    distance_thumb_index = ((thumb_tip.x - index_tip.x) ** 2 +
                            (thumb_tip.y - index_tip.y) ** 2) ** 0.5

    if (distance_thumb_index < 0.05 and
        middle_tip.y < middle_base.y and
        ring_tip.y < ring_base.y and
        pinky_tip.y < pinky_base.y):
        return "OK"


    return "UNKNOWN"


with mp_hands.Hands(
        static_image_mode=False,
        max_num_hands=2,
        min_detection_confidence=0.7,
        min_tracking_confidence=0.7) as hands:
    

    while True:
        ret, frame = cap.read()
        if not ret:
            break

        frame = cv2.flip(frame, 1)
        rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
        results = hands.process(rgb)

        gesture_text = "No Hand"

        if results.multi_hand_landmarks:
            for hand_landmarks in results.multi_hand_landmarks:
                mp_draw.draw_landmarks(
                    frame,
                    hand_landmarks,
                    mp_hands.HAND_CONNECTIONS
                )

                gesture_text = detect_gesture(hand_landmarks.landmark)

        cv2.putText(frame, gesture_text,
                    (50, 80),
                    cv2.FONT_HERSHEY_SIMPLEX,
                    1.5,
                    (0, 255, 0),
                    3)

        cv2.imshow("ASL Static Gesture Recognition", frame)

        if cv2.waitKey(1) & 0xFF == ord('q'):
            break

cap.release()
cv2.destroyAllWindows()
